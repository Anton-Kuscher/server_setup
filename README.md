# Setup for Utility Server
Running all the services required for a centralized main server.
---

**To install run** 
```bash
curl -sSL https://raw.githubusercontent.com/Anton-Kuscher/server_setup/refs/heads/master/setup.sh | bash
```
### What you get?
- Vaultwarden
- Pihole
- OpenVPN
- Homarr
- Upsnap
- My Willhaben Searchagent
